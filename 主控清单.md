# 进化中心主控清单

## 使用方法
发现问题 → 更新反馈日志 → 跑分发脚本同步到各工具

## 一键分发
```
python .进化中心/测试集/同步规则.py
```
自动同步到：
- ATOMCODE  → ~/.atomcode/ATOMCODE.md
- Hermes    → ~/AppData/Local/hermes/SOUL.md
- CodeBuddy → ~/.codebuddy/CODEBUDDY.md
- Claude Code → I:\1\CLAUDE.md
