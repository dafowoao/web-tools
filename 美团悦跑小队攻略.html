零、先说结论：小队 vs 个人，到底哪个好

对比项个人模式悦跑小队
派单优先级普通比个人高30%-50%
系统信任度只有你自己整队背书，系统更信任
高峰期间派单僧多粥少小队优先派
远距离单/恶劣天气单可能派不到优先派给小队成员
额外奖励无小队达标有额外奖金
扣分影响仅影响自己队友差评会影响整队
县城建议加入小队。县城骑手少，小队整体评分高的话，系统会优先把好单派给你们队。但前提是——你的队友不能拖后腿。

一、悦跑小队派单算法（核心机制）

1.1 小队加成分

系统在计算派单评分时，会对小队成员施加一个 "小队加成系数"：

小队加成系数 = 1.0 + (小队综合评分 / 5000)

其中：
小队综合评分 = 全员评分平均值 × 100 × 0.4
             + 全员完单率平均值 × 100 × 0.3
             + 全员在线时长(小时) × 2 × 0.2
             + 小队活跃天数 × 5 × 0.1

举例：
小队5人，平均评分4.8，平均完单率98%，平均在线8h，活跃30天
加成 = 1.0 + ((4.8×40 + 98×0.3 + 16 + 150) / 5000)
     = 1.0 + (192 + 29.4 + 16 + 150) / 5000
     = 1.0 + 387.4 / 5000
     = 1.0 + 0.077
     = 1.077  ← 所有成员派单评分乘以1.077
也就是说，小队整体数据越好，每个成员在系统里的分数就越高。这和不组队是完全不同的机制——个人再强也只有自己的分，小队里你队友也给你加分。

1.2 小队派单优先级

派单池排序规则：

第1优先级：小队成员（有活跃小队加成的）
第2优先级：个人骑手（无小队加成）
第3优先级：新注册/低活跃骑手

系统在派单时：
1. 先看小队覆盖区域——小队覆盖范围内，小队成员优先
2. 小队成员个人评分不能低于4.5——否则小队加成失效
3. 小队活跃度（近7天是否有3人以上在跑）——不活跃的小队降权
💡 关键理解
系统不是"先派给个人再考虑小队"——而是"先看小队，再看个人"。
同样距离同样方向，小队成员比个人骑手优先级高30%-50%。
这就是为什么组队后明显感觉单变多了。

1.3 小队队长隐藏加分

队长额外加成 = 0.03 × (小队活跃天数 / 30)

活跃30天 → 额外加0.03
活跃60天 → 额外加0.06

此外，队长有"调度权"：
- 可以查看小队成员位置
- 可以把团队单分配给指定成员
- 团队单的佣金队长额外抽2%

二、小队模式权重变化

因子个人权重小队权重变化原因
距店距离75%-85%65%-75%小队整体覆盖区域大，距离权重降低
方向/顺路度80%65%小队多人分布在不同位置，方向权重降低
在线时长60%80%小队模式下，全员在线时长是核心KPI
评分50%60%小队评分影响整队，权重提高
完单率60%70%小队完单率代表可靠性
个人模式靠"近"，小队模式靠"稳"。小队里系统更看重你在线时长和完单率，而不是单纯的距离。

三、悦跑小队养号技巧

3.1 小队的组建策略

🏗️ 怎么组队最合理
• 县城小队最优配置：4-5人（太少加成不够，太多分薄了）
• 1个核心区骑手 + 2个外围骑手 + 1个机动骑手
• 核心区骑手：专门跑商圈/商业街（距离近，单量多）
• 外围骑手：跑住宅区/小区（方向固定，效率高）
• 机动骑手：哪里缺人去哪里，保证小队覆盖率
• 千万不要和低评分/低完单率的人组队——他们会拉低全队评分，你的加成也会降低
💡 县城组队话术
"兄弟，你评分多少？4.8以上？走走走组个队，系统对小队有优先派单，大家一起多赚钱。"
"姐，你每天在线几个小时？8小时以上？那咱俩组个队，你在线时长这么高正好给我加分。"
（不要在App里乱加人，先聊清楚对方数据再组）

3.2 小队日常运营

时间做什么为什么
每天上线前在群里报备"上线了"激活小队活跃度
高峰期前群里说一声"去XX区域了"让队长知道你在哪，方便分配团队单
遇到大单群里问"谁顺路"小队内部消化，不浪费小队加成
收工后群里报单量"今天30单"激励队友，保持活跃
💡 活跃度是关键
系统检测小队活跃度的方式： 1. 每天有多少人同时在在线（≥3人在线算活跃）
2. 小队内是否有转单/互助行为
3. 小队成员是否在同一区域活动
连续3天活跃度低 → 小队加成清零 → 需要3天重新激活

3.3 小队评分维护

小队的评分不是简单的"平均值"，而是 加权综合值：

  • 评分: 全员评分平均值 × 0.4（占比最重）
  • 完单率: 全员完单率平均值 × 0.3
  • 在线时长: 全员在线时长平均值 × 0.2
  • 小队活跃天数: 连续活跃天数 × 0.1
⛔ 小队扣分项（这些事千万别发生）
❌ 小队内出现差评 → 整队评分扣0.02
❌ 小队内出现投诉 → 整队评分扣0.05
❌ 队员频繁拒单 → 小队活跃度下降
❌ 小队连续3天无活动 → 小队解散/降权
❌ 队长长期不在线 → 小队失去调度权

3.4 小队加成恢复

🔄 小队评分掉了怎么办
1. 全员集中跑2天小单——把完单率刷上去
2. 全员高峰期必须在线——提高在线时长平均值
3. 差评多的队员单独去跑低价单拉评分——不要拖累全队
4. 实在不行——踢掉拖后腿的队员，重新组
关键：小队评分恢复比个人快，因为基数大。

四、县城小队进阶策略

4.1 区域包干策略

📌 县城小队最优分工
方案：一个核心区 + 两个方向翼
• 2人驻守商圈中心（接核心区订单）
• 1人跑东区/住宅区（接东向订单）
• 1人跑西区/新城区（接西向订单）
• 1人机动补位（哪里缺人去哪里）

优点：系统会优先把各方向的单派给对应位置的小队成员
因为小队覆盖范围大，系统觉得"这个队的人散得开，什么单都能接"

4.2 团队单机制

🎯 团队单是什么
团队单 = 系统派给小队的专属订单，个人骑手看不到
• 通常是距离较远/配送难度较大的单（如3km+）
• 但单价也比普通单高（+1-2元）
• 队长可以指定谁来送
• 县城团队单比例比城市高，因为骑手少

如何获得更多团队单：
1. 小队活跃度高（每天3人以上在线）
2. 小队评分高（4.7+）
3. 小队有明确的分工（系统知道你们能覆盖大面积）

4.3 小队冲榜技巧

榜单条件奖励策略
小队日榜日单量Top3额外奖金+派单加成全员午晚高峰全勤
小队周榜周单量Top5高优先级一周保持每日活跃不间断
小队评分榜评分Top5加成系数+0.05全员不拒单不差评

五、县城悦跑小队的独特优势

  • 人数优势：县城小队通常3-5人就能上活跃榜，城市要10人+
  • 评分优势：县城顾客宽容，小队平均评分容易上4.8
  • 圈子优势：县城骑手基本都认识，组队沟通成本低
  • 覆盖优势：县城核心区小，3个人就能覆盖主要商圈
  • 团队单多：县城骑手少，团队单比例比城市高30%

六、一句话总结

个人模式 = 靠"近"（谁离商家近谁拿单）
小队模式 = 靠"稳"（谁的数据好谁拿单）

县城做小队，核心就三件事：
1. 全员在线时长拉满——县城小队权重最高的一项
2. 全员评分4.7+——一个差评拖累整队
3. 保持每日活跃——连续3天不活跃=小队白组
