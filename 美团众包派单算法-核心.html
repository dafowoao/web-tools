一、派单评分公式（算法核心）

系统给每个骑手算分，分最高的拿单。公式：
Score = W₁S₁ + W₂S₂ + W₃S₃ + W₄S₄ + W₅S₅ + W₆S₆ + Bias

S₁ = 距离分  — 你离商家越近分越高
S₂ = 方向分  — 送餐方向跟你当前路线一致分越高
S₃ = 负载分  — 你手上单越少分越高
S₄ = 评分分  — 你的评分越高分越高
S₅ = 在线分  — 你在线越久分越高（县城重点！）
S₆ = 完单率  — 你完成率越高分越高

四、各时段策略 + 话术

🌅 05:00-08:00 早餐

距店权重85%最高，在线时长权重70%。系统最看重"你在哪"和"你在不在线"。
  • 5:30前到早餐集中区（粥店/包子铺/肯德基）蹲着别动
  • 在线就有单——县城早间没几个骑手在线
  • 话术："早上单少，在线就有派单优势。5点起床跑早餐，虽然累但稳定。"
  • 不要锁屏！锁屏容易漏通知

☀️ 08:00-11:00 上午平峰

全程最低谷时段，适合做数据。各因子权重均衡。
  • 在商圈中心点挂机，辐射范围最大化
  • 有预约单就抢——县城商户备餐慢，预约单提前锁定
  • 话术："平峰期单价低但没竞争，系统看你在线就给你派。刷完单率和评分。跑5-8单小单把数据做漂亮。"

🔥 11:00-13:30 午高峰

全天单量最密集的时段，28%的日单量集中在这2.5小时。距店权重虽降到75%，但负载权重只有50%，系统鼓励多接。
  • 10:30前到单量TOP10商家附近蹲守——谁离店近谁先拿单
  • 接单标准：只接1.5km以内的，超过3km的直接不抢
  • 取餐时跟老板套近乎："老板还有没有顺路的？帮我看看"
  • 千万不要跑远单！高峰期跑一个3km的单=损失3-4个近单的机会
  • 目标：午高峰冲20单
  • 话术："午高峰不要贪大单，就近接、快送、再接。1km内的单秒抢，2km看方向，3km以上果断放弃。高峰期系统派单密集，送完一单通常马上派下一单。"

🌤️ 13:30-17:00 下午平峰

14:00-16:00有下午茶小高峰（奶茶/咖啡）。适合刷评分。
  • 到中学/商业街附近等下午茶单
  • 主动接一些小单提高完单率和评分
  • 话术："下午单少但没竞争。主动接那些低价小单，不是为了赚钱，是为了把完单率和评分刷上去。高评分高完单率=高峰期优先派单。"
  • 检查电动车电量，为晚高峰做准备

🌆 17:00-20:00 晚高峰

全天单价最高的时段(均价4.5+)，方向分权重最高85%。系统最关心"你送的方向对不对"。
  • 17:00前到餐饮集中区占位
  • 方向原则：送住宅区/小区方向——返程单多
  • 送工业区/偏远区容易空跑回来
  • 送完一单别急着休息，看系统是不是马上派了返回方向的单
  • 话术："晚高峰方向最重要。送到小区后别急着走，停1-2分钟等系统派返程单。县城范围小，方向对了回程就有单。往工业区送是坑——回来空跑3公里。"
  • 目标：晚高峰冲18单

🌙 20:00-23:00 夜间

骑手大量下线，在线时长权重降低，方向分权重80%居首。单价5元+。
  • 驻守烧烤/炸鸡/麻辣烫店附近
  • 3-4km的单可以接（单价高）
  • 话术："夜间骑手少，系统没得选。你在线就有派单，不用抢。距离稍远的也可以接，因为单价高。注意安全！"

🌃 23:00-02:00 深夜

基本只剩1-2个骑手，你在线≈全城单都是你的。单价6.5元最高。
  • 独享全城，不限距离
  • 注意安全！县城深夜路灯暗，提前打电话让顾客出来取
  • 话术："深夜没人跟你抢，5km的单都接。但一定要提前打电话确认顾客能出来取，别送到地方没人。"

五、四种核心场景话术

抢单判断

🧠 接到派单通知时
距店1km内 → "秒抢！系统优先给我了，因为我最近"
距店1-2km → "看方向，顺路就抢，不顺路等下一单"
距店2-3km → "高峰期不抢，平峰期看价格，4块以上考虑"
距店3km+ → "除非单价5元以上，否则放弃"
县城技巧："守在TOP5商户门口，系统优先派给最近骑手"

拒单时机

🧠 什么时候该拒
方向相反+超过1.5km → "拒！跑过去再跑回来=白干"
商家出餐巨慢 → "提前拒，不影响派单率"
送到偏远工业区/乡镇 → "拒，回程空跑"
小区不让进+大单(水/米) → "拒，搬不动"
注意：雨天拒单影响派单权重，非必要不拒

路线选择

🧠 每天该跑哪条线
核心路线：主街 → 商业街 → 住宅区（来回跑）
避开路线：老城区窄巷、工业区、学校远门
多单合并：同一商圈取2-3单再出发（省时间）
返回策略：送到小区后停门口等3分钟等返程单
电量管理：县城日跑60-100km，午高峰前充一次电

冲单技巧

🧠 怎么冲单王
午高峰11-13点 → "只接近单(1.5km内)，送完马上接下一单"
晚高峰17:30-19:30 → "单价最高，2km内都接"
平峰期 → "刷评分和完单率，不要挑单"
新手保护期(前7天) → "系统优先派单，全力冲单王！"

六、接单决策树（一图看懂接不接）

收到派单
├─ 距店 > 3km? → 不抢，等近单
│  └─ ≤3km? → 看方向
├─ 方向相反? → 不抢，除非单价>5元
│  └─ 方向顺路? → 看评分
├─ 评分 < 4.6? → 先跑几单刷分
│  └─ 评分 ≥ 4.6? → 看时段
├─ 高峰期? → 只接<2.5km近单
│  └─ 平峰期? → 可接3-4km
└─ 深夜? → 全城独享
